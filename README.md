# vfpbenchmark

Runs benchmark tests for the FPU and SIMD units.


### Linux

```bash
$ git clone https://github.com/hiroog/FlatBuildTool
$ export PATH=$PATH:$PWD/FlatBuildTool
$ git clone https://github.com/hiroog/vfpbench
$ cd vfpbench
$ flmake
$ flmake run
$ more output_log.txt
$ flmake list
```

### macOS

macOS is not supported.


### Android + termux

```bash
$ pkg install git python clang
$ git clone https://github.com/hiroog/FlatBuildTool
$ export PATH=$PATH:$PWD/FlatBuildTool
$ git clone https://github.com/hiroog/vfpbench
$ cd vfpbench
$ flmake
$ flmake run
$ more output_log.txt
$ flmake list
```


### Windows

If you want to run this on Windows, please use WSL1.
